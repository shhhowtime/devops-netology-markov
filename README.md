-Все файлы, которые содержат .tfstate. в середине

-Файл crash.log

-Все файлы, заканчивающиеся на .tfvars

-Файл override.tf

-Файл override.tf.json

-Все файлы, заканчивающиеся на _override.tf

-Все файлы, заканчивающиеся на _override.tf.json
